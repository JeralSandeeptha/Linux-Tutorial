(1.) Select a word
   * Copy something   
 
(2.) Right Click 
   * Paste something 

(3.) Zoom In Terminal
   * CTRL + SHIFT + +

(4.) Zoom Out Terminal
   * CTRL + -

(5.) Stop running process
   * CTRL + C

(6.) Clear Terminal
   * CTRL + L

(7.) List folder items
   * ls

(8.) List folder items in detail
   * ls -a
