(1.) pwd    
   * To get the current directory path  
 
(2.) ls    
   * Show the things in the folder   

(3.) ls - a    
   * Show the things in the folder including hidden files

(4.) ls -lh   
   * Show the things in the folder with details 

(4.) ls -lR foldername/
   * List with sub direcrotries

(5.) cd / 
   * Go to the root directory 

(6.) cd .. 
   * Go to back folder

(7.) cd 
   * Go to next folder

(8.) clear
   * clear the console / looks as new console

(9.) mkdir 
   * To create a new folder / directory
